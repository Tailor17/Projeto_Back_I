Plataforma de e-commerce simples onde existe apenas um vendedor, que disponibiliza semanalmente os produtos disponíveis, previamente cadastrados no banco de dados, e recebe os pedidos feitos pelos clientes com endereço de entrega, preço e itens. Já no lado do cliente temos uma interface simples com um carrinho de compras, onde seleciona os produtos desejados para a entrega e finaliza o pedido utilizando seu endereço cadastrado ou um novo endereço. O cliente também tem a possibilidade de editar as suas informações padrões.
